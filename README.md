
# Address Book Application

This is a simple Spring Boot based Address Book application built to manage contacts efficiently.
It provides basic CRUD operations using REST APIs and connects to a PostgreSQL database for data storage.

The project follows a clean layered architecture with:

* Controller layer
* Service layer
* Repository layer
* DTO and Exception handling


## Tech Stack

* Java 21
* Spring Boot 3
* Spring Data JPA
* PostgreSQL
* Hibernate
* Maven


## Features

* Add a new contact
* View all contacts
* Get contact by ID
* Update contact details
* Delete a contact
* Input validation and global exception handling
* Swagger UI for API testing


Application will start on:

```
http://localhost:8080
```

Swagger UI:

```
http://localhost:8080/swagger-ui/index.html
```
